# Related Work Prompt

Use this to compare papers without drifting into vague summary.

## Prompt

```text
Compare these papers for a thesis related-work section.
Focus on:
- problem setting,
- method,
- evidence,
- limitations,
- what gap remains for my thesis.

Return:
1. a comparison table,
2. a paragraph-sized synthesis,
3. claims that still need direct source checking.

Papers and notes:
[PASTE MATERIAL]
```
