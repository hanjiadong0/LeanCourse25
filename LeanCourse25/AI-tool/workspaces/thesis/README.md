# Thesis Workspace

This workspace is the default AI-assisted staging area for thesis writing.

Use it for:

- chapter planning
- literature review extraction
- argument structure and revision passes
- experiment-to-writing translation
- daily notes before a dedicated thesis repo exists

If you later move thesis work into a real repository, keep this workspace as the lightweight prompt and process layer, then set `thesis.project_path` in `config\profiles.local.json`.
